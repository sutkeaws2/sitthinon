ผมเป็น Backend Developer และ Fontend ได้เล็กน้อย กำลังศึกษาอยู่ในมัธยมศึกษาปีที่ 4 โรงเรียนนครไทยนี้เป็น Web-Site แนะนำตัวของผม

เกี่ยวกับผม

สวัสดีครับผมชื่อเล่นคาสตรอลเด็กมัธยมกำลังหาประสบการณ์มีส่วนไปช่วยงานด้าน Backend Development มาหลายงานแล้วครับจัดทำเว็บไซต์ตัวนี้ขึ้นเพื่อแสดงผลงานครับผมและทำให้คนรู้จักผมมากขึ้น ส่วนตัวผมเริ่มเรียนรู้ภาษาโปรแกรมตั้งแต่ช่วงมัธยมศึกษาตอนต้นอย่างจริงจังโดยเริ่มที่ศึกษาภาษา Python และ Java ด้วยตัวเองทั้งนั้น ผมอยากนำความรู้ที่ได้เรียนรู้มาใช้ในการพัฒนาเว็บไซต์และแอปพลิเคชันที่มีประโยชน์ตอนนี้ประสบการณ์อาจจะยังน้อยแต่ฝีมือผมเกินร้อยครับ สามารถติดต่อร่วมงานได้ครับ
